# 普通字符

[abc] 匹配[...]的所有字符

[^abc] 取反，除了[...]的其他字符

[A-Z] 区间字母A到Z

**.** 匹配除（\\n换行符 \\r 回车符）的任何单个字符

**\\s** **\\S** 匹配所有，**\\s**所有空白符，包括换行 ;  **\\S**非空白符，不包括换行

**\\w** 匹配字母、数字、下划线

# 特殊字符

 **$** ：结尾位置 （）

**^** ：开始位置（）

**()** ：子表达式开始和结束（匹配字符----\(   和  \））

**\*** ：匹配**零次**或多次（）

**\+** ：匹配**一次**或多次（）

**?** ：匹配零次或一次（）

**|** ：表示或，两项间的一个（匹配字符----\|）
