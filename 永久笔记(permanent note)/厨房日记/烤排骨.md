# 慢烤牛肋排
调味腌制（可过夜）
- 黑胡椒粉，蒜粉，洋葱粉 1：1
- 盐（盐蒜粉比例 2:3）
- 红糖（盐糖比 1:7）
准备酱料
- 胡椒粉 5g，蒜粉 15 g，盐 3 g
- 烟熏辣椒粉 15g
- 蕃茄酱黄芥末酱各 50 克
- 红糖 50 克
- 苹果醋和水各自 75 克混合
烤制分四阶段，每阶段 45 分钟5 130 ℃（265 ℉）
1. 裸烤
2. 正面刷酱料
3. 反面刷酱料翻面
4. 正面刷酱料再翻面
