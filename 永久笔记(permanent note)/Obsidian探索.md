# 快捷键记录
## 常用和好用的快捷键
### 常用功能
1. **Ctrl+P**：打开命令栏
2. **Ctrl+ ，**：打开*设置面板*
3. **Ctrl+Alt+D**：打开今日日记
4. **Alt+N**：使用 templater 创建笔记
### 编辑
1. **Ctrl + ;**：为当前笔记快速添加 property
2. **Ctrl+'**：快速添加tag
### 窗口操作
1. **Ctrl+nums**：切换到指定标签页
2. **Ctrl+T（Tab）**：打开新的标签页
3. **Ctrl+W**：关闭当前 page
4. **Ctrl+alt+W**：关闭其他所有标签页
5. **Ctrl+alt+➡/⬅**：前进/后退#
### 文件切换
1. **ctrl+O**：快速切换（从当前窗口笔记切换到其他笔记）
