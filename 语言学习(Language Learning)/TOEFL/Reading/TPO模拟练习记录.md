# 托福阅读5步练习法
1. 限时做题，标记超过3分钟没有答出来的题（每一篇不超过18分钟），*目的是模拟考试环境*
2. 重做限时做时没把握的题，*目的是确保对问题和文章的理解能力，答题能力*
3. 对答案，分析错题原因
4. 精读，整理段落结构（建议用思维导图）,*着重精读和题目相关的部分*
5. 重读文章，用能真正理解的最快速度，保证7分半内读完
