#英语/单词后缀
 # 英语中后缀决定了一个词的词性，它决定了这个词是**动词**，**形容词**，**名词**还是**副词**


# 1. “-able” 
## 【形容词后缀】：能......的；具有......性质的

# 2. “-al” 
## 【形容词后缀】：具有......性质的；属于......的
## 【名词后缀】：表示抽象名词，表示人

# 3. “-ate” 
## 【动词后缀】：做；造成
## 【名词后缀】：抽象名词，表示人

# 4. “-ed” 
## 【形容词后缀】：加在名词后：有......的；加在动词后：已......的，被......的

# 5. “-en” 
## 【动词后缀】：使变成.....的

# 6. “-ence” 
## 【名词后缀】：抽象名词

# 7. “-ful” 
## 【形容词后缀】：富有......的；具有......性质的

# 8. “-fy” 
## 【动词后缀】：使变成......的样子

# 9. “-ic” 
## 【形容词后缀】：......的

# 10. “-ion” 
## 【名词后缀】：抽象名词
