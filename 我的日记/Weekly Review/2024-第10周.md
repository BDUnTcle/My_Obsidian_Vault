# 本周日记
```dataview
Table file.cday as "创建日期", file.mday as "最后修改"
From "我的日记" AND #日记 
WHERE file.cday >= date(today) - dur(1 week)
SORT date ASCENDING
```
# 本周课堂笔记
```dataview
Table file.cday as "创建日期", file.mday as "最后修改"
From "瞬时笔记(fleetting note)" 
WHERE type = "CourseNotes" AND  file.cday >= date(today) - dur(1 week)
SORT date ASCENDING
```
